# Darkeye
Search engline for legal websites in darknet (the dark web).

- To run the Darkeye server:

```
$ ./run
```

`run` will automatically call `build` if needed.

- To update then run:

```
$ ./run update
```

- To run with full update

```
$ ./run update full
```

- To just build:

```
$ ./build
```

- To update packages:

```
$ ./update
```

- To clean the build:

```
$ ./clean
```

- To create self signed certificate:

```
$ ./create-certificate
```
